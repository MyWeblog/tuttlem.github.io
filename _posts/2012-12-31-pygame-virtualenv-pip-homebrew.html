I finally just got an installation up and running on my MBP for pygame inside of virtualenv. It took a little bit of investigation, but the following should get anyone out of the installation rut (if they're interested in keeping their environments isolated).

First, all of your SDL stuff should be installed before going any further. If it's not, it's as easy as this from the command shell.

$ brew install sdl sdl_image sdl_mixer sdl_ttf smpeg portmidi

You're almost done. Create your virtual environment and switch into it. Once you've done that, you just need to issue the following and you're done.

$ pip install hg+http://bitbucket.org/pygame/pygame

That will grind away and before you know it, you're ready to go.

Then it's onto reading The Python Game Book, The Line-by-Line Chimp Example and any other tutorials you can get your hands on.
